### 🚀 Quick Start Example

This folder contains a sample animation created in DragonBones and a ready-to-use Defold project.

To get started, download and unzip the following files:

- **`Dragonboy Dragonbones Project.zip`**: Contains the original DragonBoy mascot animations and project files.
- **`DB Reborn 1.0.0 Exported JSON.zip`**: Contains the JSON exported from DragonBones, the image library, and the `Dragonboy.spinejson` file generated by DB Reborn 1.0.0.
- **`Defold 1.12.0 Dragonboy Project Example.zip`**: A complete, working Defold project featuring the DragonBoy animation.

### ⚠️ Important Notes

To ensure everything works as expected, please note the versions used in this example:

- **DragonBones Version:** 5.6.2.0001
- **Defold Version:** 1.12.0
- **Spine Extension for Defold:** 4.2.0

**Note:** If you update Defold or the Spine Extension to a newer version, the project might require adjustments to remain compatible.
